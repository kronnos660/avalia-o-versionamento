# 🌐 Estudos Front-End

## 📝 Descrição do Projeto/Atividade

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento Front-End. A atividade consistiu na criação de uma página web utilizando HTML, explorando a estrutura básica de um site e a organização dos elementos visuais.

O projeto serviu como introdução ao desenvolvimento web, permitindo compreender como as páginas são construídas e exibidas nos navegadores.

---

## 🧠 Reflexão de Aprendizado

### 1. O que aprendi?

Durante a realização desta atividade, aprendi:

* Estrutura básica de um documento HTML.
* Utilização de tags semânticas.
* Organização de conteúdo em páginas web.
* Importância da hierarquia dos elementos.
* Funcionamento dos navegadores na renderização de páginas.
* Conceitos iniciais de desenvolvimento Front-End.

Além disso, compreendi como criar páginas organizadas e de fácil manutenção.

### 2. Para que serve (Por que aprendi)?

O desenvolvimento Front-End é responsável pela parte visual dos sistemas e aplicações web.

Aprender HTML é fundamental porque:

* É a base de qualquer página web.
* Permite estruturar conteúdos de forma organizada.
* É utilizado em conjunto com CSS e JavaScript.
* É uma das competências mais exigidas na área de tecnologia.
* Facilita a criação de interfaces para usuários.

## 💻 Demonstração e Como Rodar

### Código Relevante Comentado

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
</head>
<body>
    <h1>Olá Mundo!</h1>
    <p>Primeira página criada utilizando HTML.</p>
</body>
</html>
```

### Explicação

* `<!DOCTYPE html>` define o documento como HTML5.
* `<head>` contém informações da página.
* `<body>` contém o conteúdo exibido ao usuário.
* `<h1>` representa o título principal.
* `<p>` representa um parágrafo.
